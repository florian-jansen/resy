# Show expert classification priority, abbreviation and name

This function shows the type definitions given in the chosen ESY system.

## Usage

``` r
resy_show_vegtypes(expert, width = 80, indent = "  ")
```

## Arguments

- expert:

  Result of \`resy_load_expert\`

- width:

  maximum number of characters shown

- indent:

  characters used for indentation of the type levels

## Value

Prints only to the console
